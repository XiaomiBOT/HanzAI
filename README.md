# HanzAI - Advanced AI Chatbot

HanzAI adalah chatbot AI berbasis web yang dirancang mirip dengan Gemini. Menyediakan interface modern dan interaktif untuk percakapan dengan AI.

## 🌟 Fitur

- 💬 Real-time chat interface
- 🤖 Advanced AI responses
- 📝 Conversation history
- 🎨 Modern UI design (mirip Gemini)
- 🔐 User authentication
- 💾 Persistent storage
- 🚀 Fast & responsive

## 📋 Tech Stack

### Backend
- Python 3.9+
- FastAPI
- SQLAlchemy
- PostgreSQL/SQLite

### Frontend
- React 18+
- Tailwind CSS
- Axios
- Socket.io (real-time)

## 🚀 Instalasi & Setup

### Prerequisites
- Node.js 16+
- Python 3.9+
- Git

### Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
